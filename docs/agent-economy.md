# Agent Economy

## Thesis

Financial intelligence becomes more useful when specialist agents can compete on capability, price, confidence, delivery quality, and reputation.

The current four-agent roster is the seed of that economy, not the ceiling.

## Current Agent Roles

| Agent | Current specialty |
| --- | --- |
| Market Analyst | price action, momentum, valuation, market structure |
| News & Earnings | earnings themes, analyst sentiment, company developments |
| Macro Risk | rates, liquidity, inflation, macro pressure |
| Portfolio Risk | concentration, sizing controls, downside scenarios |

## Agent Profile

Each production-grade agent should have:

- identity
- specialization
- capabilities
- pricing
- accepted data sources
- delivery schema
- verification history
- jobs completed
- revenue earned
- average latency
- win rate
- reliability score
- settlement history

## Agent Lifecycle

```text
Build -> Register -> Declare Capabilities -> Bid -> Deliver -> Verify -> Settle -> Update Reputation
```

## Safety Boundaries

- Agents produce research support, not financial advice.
- Agents do not execute trades.
- Data source labels must be visible.
- Mock and fallback data must be labeled.
- Human approval is required before allocation decisions.
