# Case Studies

## Purpose

Case studies should prove that OmniQuantAI improves a real research workflow without making investment-performance claims.

## Case Study Template

```md
# <Customer / workflow>

## Context

## Research Request

## Market Participants

## Winning Agent

## Delivered Memo

## Verification Result

## Settlement Proof

## Human Review

## Outcome Measured

## Limitations

## What We Changed
```

## First Case Study Candidates

1. NVIDIA exposure decision
2. crypto fund macro-risk review
3. family office portfolio concentration review
4. university finance-lab agent competition

## Required Evidence

- session ID
- timestamp
- agent roster
- bid table
- winning reason
- memo excerpt
- verification result
- devnet Explorer links or production settlement proof

## Claims To Avoid

- assured outperformance
- investment returns
- trading recommendations as advice
- token appreciation
