# Public Launch Plan

## Launch Goal

Move OmniQuantAI from a private prototype to a credible open-source Financial Intelligence Network project.

## Launch Prerequisites

- clean README
- working `npm run judge`
- fresh devnet evidence session
- demo video
- settlement Explorer links
- agent builder guide
- public roadmap
- design-partner page
- media kit

## Launch Sequence

1. Tag release.
2. Publish GitHub release notes.
3. Publish demo video.
4. Post technical launch article.
5. Share X thread.
6. Share LinkedIn post.
7. Open GitHub Discussions or a public feedback channel.
8. Invite design partners.
9. Invite developers to build specialist agents.

## Launch Narrative

Lead with proof:

```text
An AI buyer agent requested financial intelligence, specialist seller agents competed, the winner delivered an Investment Committee Memo, and Solana devnet escrow released payment.
```

## Day-Of Checklist

- repo public and clean
- CI passing
- no secrets committed
- demo video link works
- README quickstart works
- design-partner CTA ready
- media kit ready

## Post-Launch Follow-Up

Within 48 hours:

- respond to issues
- create good first issues from feedback
- publish a technical breakdown
- schedule office hours
- record launch metrics
