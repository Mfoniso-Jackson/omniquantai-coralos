# Ecosystem Roadmap

## Roadmap Principle

Expand only when the previous market is reliable, observable, and useful. The core lifecycle remains sacred:

```text
WANT -> BID -> AWARD -> DEPOSITED -> DELIVERED -> VERIFIED -> RELEASED
```

## Phase 0: Bootstrap Market

Current:

- one buyer
- four sellers
- NVDA research request
- Solana devnet settlement
- deterministic verifier
- dashboard proof
