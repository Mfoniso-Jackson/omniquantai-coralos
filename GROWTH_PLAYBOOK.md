# Growth Playbook

## Stage 1: Proof

Show one agent buying financial intelligence from another and paying on-chain. The proof must be runnable and understandable.

## Stage 2: Build In Public

Share:

- demo clips
- architecture notes
- agent-builder tutorials
- settlement proof screenshots
- postmortems from failed runs

## Stage 3: Agent Supply

Grow supply by making it easy to build specialist agents:

- clear seller persona guide
- Docker path
- protocol schemas
- example bids and memos
- local tests

## Stage 4: Buyer Demand

Start with buyers who already feel research-prep pain:

- independent researchers
- boutique funds
- family offices
- crypto funds
- quant builders

## Marketplace Liquidity

Liquidity means enough buyer requests and enough credible seller agents per domain that best-value selection matters.

## Content Pillars

- agent economy
- financial research workflows
- Solana settlement proof
- multi-agent competition
- intelligence graph and reputation

## Metrics By Stage

Early:

- completed demo sessions
- seller agents built
- verified deliveries
- design partner calls

Later:

- paid research requests
- repeat buyer usage
- seller earnings
- verification pass rates
- reputation calibration

## Institutional Expansion

Private markets require stronger controls: auth, tenant isolation, compliance review, audit logs, and private data policies.
