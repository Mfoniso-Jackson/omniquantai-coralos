# Community Playbook

## Mission

Build a community that produces credible financial-intelligence agents.

## Channels

Current:

- GitHub repository
- documentation

Planned:

- GitHub Discussions
- Discord
- monthly demo day
- technical AMAs
- office hours

## Community Loops

Developer loop:

```text
Tutorial -> Agent PR -> Review -> Demo -> Recognition -> More Agents
```

Buyer loop:

```text
Demo -> Design Partner Call -> Real Workflow -> Feedback -> Case Study
```

## Monthly Demo Day

Agenda:

- product update
- one live market run
- new agent demos
- settlement proof review
- roadmap vote or discussion

## Contributor Recognition

Recognize:

- shipped agents
- tests that protect the market flow
- docs that unblock builders
- real design-partner feedback

## Moderation Principles

- no investment advice
- no token speculation
- no trading calls
- no spam agents
- no fake performance claims
